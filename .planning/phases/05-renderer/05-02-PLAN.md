  <action>
    **DebugRenderer.ts** — Static debug drawing utility:
    - Create a class or namespace of static methods that draw debug overlays onto a Canvas 2D context. The methods receive the context, data, and scale factor.
    - All drawing uses 1px line width adjusted for scale: `lineWidth = 1 / scale`.

    `drawAABBs(ctx, bodies: readonly Body[], scale: number)`:
    - Set `ctx.strokeStyle = '#00ff00'` (green), `ctx.lineWidth = 1 / scale`.
    - For each body: compute AABB via `body.shape.computeAABB(body.position, body.angle)`.
    - Draw `ctx.strokeRect(aabb.min.x, aabb.min.y, aabb.max.x - aabb.min.x, aabb.max.y - aabb.min.y)`.

    `drawContacts(ctx, manifolds: readonly Manifold[], scale: number)`:
    - For each manifold, for each contact point:
      - Draw red filled circle at `c.point`: `ctx.fillStyle = '#ff0000'`, `ctx.arc(c.point.x, c.point.y, 3 / scale, 0, Math.PI * 2)`, `ctx.fill()`.

    `drawNormals(ctx, manifolds: readonly Manifold[], scale: number)`:
    - For each manifold, for each contact point:
      - Draw blue line from `c.point` in direction of `manifold.normal`, length `0.5` meters.
      - `ctx.strokeStyle = '#0088ff'`, `ctx.lineWidth = 1.5 / scale`.
      - `ctx.moveTo(c.point.x, c.point.y)`, `ctx.lineTo(c.point.x + m.normal.x * 0.5, c.point.y + m.normal.y * 0.5)`.

    `drawConstraints(ctx, constraints: readonly Constraint[], scale: number)`:
    - Set `ctx.strokeStyle = '#ffdd00'` (yellow), `ctx.lineWidth = 1.5 / scale`.
    - For each constraint:
      - Get `anchorA = constraint.getWorldAnchorA()`, `anchorB = constraint.getWorldAnchorB()`.
      - Draw line from `(anchorA.x, anchorA.y)` to `(anchorB.x, anchorB.y)`.
      - Draw small filled circles at each anchor point (radius `2 / scale`).

    **Renderer.ts** — Add debug rendering pass:
    - Import `DebugRenderer` from `./DebugRenderer.js`.
    - Add a `debug` boolean property (from options) and a public setter: `setDebug(enabled: boolean): void`.
    - In `draw(alpha)`, AFTER drawing all bodies, if `this.debug` is true:
      - Get `manifolds = this.world.getManifolds()`.
      - Get `constraints = this.world.getConstraints()`.
      - Get `bodies = this.world.getBodies()`.
      - Call `DebugRenderer.drawAABBs(ctx, bodies, this.scale)`.
      - Call `DebugRenderer.drawContacts(ctx, manifolds, this.scale)`.
      - Call `DebugRenderer.drawNormals(ctx, manifolds, this.scale)`.
      - Call `DebugRenderer.drawConstraints(ctx, constraints, this.scale)`.

    **index.ts** — Add DebugRenderer export:
    - Add `export { DebugRenderer } from './DebugRenderer.js';`
