    **RenderOptions.ts** — Configuration types:
    - Export `RenderOptions` interface with fields:
      - `width?: number` — canvas width in CSS pixels (default: canvas.clientWidth or 800)
      - `height?: number` — canvas height in CSS pixels (default: canvas.clientHeight or 600)
      - `scale?: number` — pixels per meter (default: 50)
      - `offsetX?: number` — horizontal offset in pixels (default: width / 2, centering origin)
      - `offsetY?: number` — vertical offset in pixels (default: 50, origin near bottom)
      - `background?: string` — background color (default: '#1a1a2e')
      - `debug?: boolean` — enable debug overlays (default: false). Placeholder for Plan 02.
      - `showRotation?: boolean` — show rotation indicators on circles (default: true)
    - Export `DEFAULT_RENDER_OPTIONS` with the above defaults.
    - Export a body color palette: `BODY_PALETTE: readonly string[]` — 8 visually distinct semi-transparent colors for palette cycling. Example: `['rgba(69,179,224,0.7)', 'rgba(239,83,80,0.7)', 'rgba(102,187,106,0.7)', 'rgba(255,167,38,0.7)', 'rgba(171,71,188,0.7)', 'rgba(255,241,118,0.7)', 'rgba(77,208,225,0.7)', 'rgba(255,112,67,0.7)']`.
    - Export `STATIC_BODY_COLOR = 'rgba(120,120,130,0.6)'` and `STATIC_BODY_STROKE = '#555'`.

    **Renderer.ts** — Core renderer class:
    - Import `World`, `Body`, `BodyType`, `ShapeType`, `Circle`, `Polygon` from the library.
    - Import `lerp` from `../math/utils.js`.
    - Import `RenderOptions`, `DEFAULT_RENDER_OPTIONS`, `BODY_PALETTE`, `STATIC_BODY_COLOR`, `STATIC_BODY_STROKE`.

    Constructor: `constructor(canvas: HTMLCanvasElement, world: World, options?: Partial<RenderOptions>)`
    - Merge options with defaults. Store `canvas`, `ctx = canvas.getContext('2d')!`, `world`.
    - Set `canvas.width` and `canvas.height` from options (for crisp rendering).
    - Store resolved `scale`, `offsetX`, `offsetY`, `background`, `showRotation`.
    - Initialize `running = false`, `lastTime = 0`.

    Public API:
    - `start(): void` — set `running = true`, call `requestAnimationFrame(this.loop)`.
    - `stop(): void` — set `running = false`.
    - `draw(alpha: number): void` — public for manual render-only usage (no rAF).

    Private `loop = (now: number): void` — arrow function for stable `this`:
    - If not running, return.
    - If `lastTime === 0`, set `lastTime = now` and schedule next frame (skip first frame to avoid large dt).
    - Compute `frameDt = (now - lastTime) / 1000`. Clamp to max 0.1s.
    - Set `lastTime = now`.
    - Call `const alpha = this.world.step(frameDt)`.
    - Call `this.draw(alpha)`.
    - Schedule `requestAnimationFrame(this.loop)`.

    `draw(alpha: number)`:
    - Clear canvas with background color (fillRect full canvas in screen space BEFORE setting the transform).
    - Set up Y-up coordinate transform: `ctx.setTransform(scale, 0, 0, -scale, offsetX, canvas.height - offsetY)`.
    - Set default line width: `ctx.lineWidth = 1 / scale` (1px regardless of scale).
    - Get bodies from `this.world.getBodies()`.
    - For each body, compute interpolated position and angle:
      - If body is static: use `body.position.x`, `body.position.y`, `body.angle` (no interpolation).
      - Else: `renderX = lerp(body.prevPosition.x, body.position.x, alpha)`, same for Y and angle.
    - Set fill/stroke colors: static bodies get STATIC_BODY_COLOR/STATIC_BODY_STROKE. Dynamic/kinematic get `BODY_PALETTE[body.id % BODY_PALETTE.length]` for fill, a slightly darker stroke.
    - Dispatch on `body.shape.type`:
      - `ShapeType.Circle`: call `drawCircle(ctx, body.shape as Circle, renderX, renderY, renderAngle)`.
      - `ShapeType.Polygon`: call `drawPolygon(ctx, body.shape as Polygon, renderX, renderY, renderAngle)`.

    `drawCircle(ctx, circle, x, y, angle)`:
    - `ctx.save()`, `ctx.translate(x, y)`, `ctx.rotate(angle)`.
    - Apply shape offset: `ox = circle.offset.x`, `oy = circle.offset.y`.
    - `ctx.beginPath()`, `ctx.arc(ox, oy, circle.radius, 0, Math.PI * 2)`, `ctx.fill()`, `ctx.stroke()`.
    - If `showRotation`: draw line from `(ox, oy)` to `(ox + circle.radius, oy)` with current stroke style.
    - `ctx.restore()`.

    `drawPolygon(ctx, polygon, x, y, angle)`:
    - `ctx.save()`, `ctx.translate(x, y)`, `ctx.rotate(angle)`.
    - Get `polygon.vertices`. Apply shape offset.
    - `ctx.beginPath()`, `moveTo(verts[0].x + ox, verts[0].y + oy)`, loop `lineTo` for remaining vertices.
    - `ctx.closePath()`, `ctx.fill()`, `ctx.stroke()`.
    - `ctx.restore()`.
