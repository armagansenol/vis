**src/math/utils.ts** (MATH-04):
- Export `EPSILON = 1e-6`
- `clamp(value, min, max)`: returns value clamped to [min, max]
- `lerp(a, b, t)`: linear interpolation a + (b - a) * t
- `approxEqual(a, b, epsilon = EPSILON)`: Math.abs(a - b) < epsilon
- `degToRad(deg)`: deg * Math.PI / 180
- `radToDeg(rad)`: rad * 180 / Math.PI
- `randomRange(min, max)`: random float in [min, max)

**src/math/Vec2.ts** (MATH-01):
Follow the RESEARCH.md "Mutable Vec2 with Method Chaining" pattern exactly. Class with public x, y fields.

Instance methods (mutable, return `this`): add, sub, scale, normalize, rotate, perpendicular, set, negate, copy(other).
Instance methods (non-mutating, return scalar): dot, cross, length, lengthSquared.
Instance method (returns new Vec2): clone.

Static methods: Vec2.add(a,b), Vec2.sub(a,b), Vec2.dot(a,b), Vec2.cross(a,b), Vec2.distance(a,b), Vec2.distanceSquared(a,b).
Static factories: Vec2.zero(), Vec2.one(), Vec2.fromAngle(radians).

**src/math/Mat2.ts** (MATH-02):
Minimal API per RESEARCH.md recommendation. Store 4 values (m00, m01, m10, m11).
- `Mat2.fromAngle(radians)`: creates rotation matrix [cos,-sin; sin,cos]
- `setAngle(radians)`: mutates in place
- `mulVec2(v: Vec2): Vec2`: returns new Vec2 = matrix * v (does NOT mutate v)
- `transpose(): Mat2`: mutates in place, returns this (transpose of rotation = inverse rotation)
- `static identity()`: returns identity matrix

**src/math/AABB.ts** (MATH-03):
Follow RESEARCH.md "AABB Overlap Test and Combine" pattern.
- Constructor takes min: Vec2, max: Vec2
- `overlaps(other: AABB): boolean` — axis-separation test
- `static combine(a, b): AABB` — enclosing AABB
- `contains(other: AABB): boolean`
- `width()`, `height()`, `center()` getters
- `perimeter(): number` — useful for broadphase heuristics later

**src/math/index.ts**: Barrel export all math types and functions.

**Tests** — write comprehensive tests with known analytical values:
- Vec2: chain operations, verify dot/cross against manual computation, normalize unit vector, rotate 90 degrees, perpendicular, static factories
- Mat2: rotation of (1,0) by pi/2 should give (0,1), transpose of rotation, identity
- AABB: overlapping pair, non-overlapping pair, containment, combine
- utils: clamp at boundaries, lerp at 0/0.5/1, approxEqual with near-epsilon values, angle conversions round-trip
