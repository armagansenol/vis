1. **ManifoldMap** (`src/collision/ManifoldMap.ts`):
   - Class stores `manifolds: Map<string, Manifold>` keyed by `pairKey(bodyA.id, bodyB.id)`.
   - Method `update(newManifolds: Manifold[]): { began: Manifold[]; ended: Manifold[]; active: Manifold[] }`:
     a. Create a new map from newManifolds keyed by pairKey.
     b. **began**: pairs in new map but NOT in old map -> these are new collisions.
     c. **ended**: pairs in old map but NOT in new map -> these collisions stopped.
     d. **warm-start transfer**: for pairs in BOTH maps, match new contacts to old contacts by feature ID (`contact.id`). Copy any `normalImpulse` and `tangentImpulse` fields from old to new (these fields default to 0 on new contacts, Phase 3 solver will populate them). Add `normalImpulse: number` and `tangentImpulse: number` to ContactPoint interface (default 0).
     e. Replace old map with new map.
     f. Return `{ began, ended, active: Array.from(newMap.values()) }`.
   - Method `clear()`: empties the map.

2. **EventDispatcher** (`src/events/EventDispatcher.ts`):
   - Export interface `ContactEvent { bodyA: Body; bodyB: Body; manifold: Manifold; isSensor: boolean; }`.
   - Export type `ContactListener = (event: ContactEvent) => void`.
   - Class `EventDispatcher`:
     - `private beginListeners: ContactListener[]`, `private endListeners: ContactListener[]`.
     - `onBeginContact(listener: ContactListener): () => void` -- returns unsubscribe function.
     - `onEndContact(listener: ContactListener): () => void` -- returns unsubscribe function.
     - `emit(type: 'begin' | 'end', event: ContactEvent): void`.
     - `clear()`: removes all listeners.
   - Keep it lightweight -- no Node.js EventEmitter dependency.

3. **Barrel exports**: Create `src/events/index.ts` exporting EventDispatcher and ContactEvent.

4. **Tests** (`tests/collision/ManifoldMap.test.ts`):
   - First update with one manifold -> began contains it, ended empty.
   - Second update with same pair -> began empty, ended empty, active contains updated manifold.
   - Second update WITHOUT the pair -> ended contains it.
   - Multiple pairs: add 3, remove 1 -> began shows 0, ended shows 1.
   - Warm-start transfer: set normalImpulse on old contact, update with matching feature ID -> new contact has impulse values.

5. **Tests** (`tests/events/EventDispatcher.test.ts`):
   - Register listener, emit event -> listener called with correct data.
   - Multiple listeners -> all called.
   - Unsubscribe -> listener no longer called.
   - clear() -> no listeners fire.

1. **CollisionSystem** (`src/collision/CollisionSystem.ts`):
   - Constructor accepts optional `{ cellSize?: number }` config.
   - Owns: `SpatialHash`, `ManifoldMap`, `EventDispatcher`, `pairExclusions: Set<string>` (empty for now, Phase 4 populates it for jointed bodies).
   - Method `detect(bodies: Body[]): Manifold[]`:
     a. Clear spatial hash.
     b. For each body: compute AABB via `body.shape.computeAABB(body.position, body.angle)`, insert into spatial hash.
     c. Query candidate pairs from spatial hash with `shouldCollide` filter.
     d. Also filter out pairs in `pairExclusions` set.
     e. For each pair: call `detectNarrowphase(bodyA, bodyB)`. Collect non-null manifolds.
     f. Update ManifoldMap with new manifolds -> get began/ended/active.
     g. Fire `beginContact` events for each began manifold.
     h. Fire `endContact` events for each ended manifold.
     i. Return active manifolds (for solver consumption in Phase 3).
   - Expose `onBeginContact` and `onEndContact` as delegate methods to internal EventDispatcher.
   - Method `addPairExclusion(idA: number, idB: number)` and `removePairExclusion(idA: number, idB: number)` for Phase 4 forward-compatibility.

2. **Update barrel exports**:
   - `src/collision/index.ts`: export everything (SpatialHash, CollisionFilter, narrowphase, sat, Manifold, ManifoldMap, CollisionSystem).
   - `src/index.ts`: ensure `export * from './collision/index.js'` and `export * from './events/index.js'` are present.

3. **Integration tests** (`tests/collision/CollisionSystem.test.ts`):

   - **Persistence**: Detect same pair twice -> no beginContact on second call, manifold updated in place.
   - **Sensor handling**: Create sensor body overlapping normal body -> manifold.isSensor = true, event fires with isSensor = true.

   - **Pair exclusion**: Add pair exclusion -> no manifold for that pair.
   - **Mixed shapes**: Circle vs polygon detection through full pipeline.
