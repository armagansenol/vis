   - Constructor accepts `cellSize: number = 2`. Store `cellSize` and `invCellSize = 1 / cellSize`.
   - Internal `cells: Map<number, Body[]>` for storing bodies per cell.
   - `clear()`: Clears the cells map.
   - `insert(body: Body, aabb: AABB)`: Compute cell range from AABB min/max using `Math.floor(coord * invCellSize)`. Loop cx/cy, compute cell key using hash function `((cx * 92837111) ^ (cy * 689287499)) | 0`, push body into cell array.
   - `queryPairs(filter: (a: Body, b: Body) => boolean): Array<[Body, Body]>`:
     - Iterate all cells. For each cell with 2+ bodies, check all pairs.
     - Deduplicate using a `Set<string>` with pair key `min(idA,idB):max(idA,idB)`.
     - Apply the filter function (pass `shouldCollide`) to each candidate pair.
     - Return array of unique pairs that pass the filter.
   - Export the class.
